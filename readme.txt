README
------

This project allows you to run Cassini as a service.
More info on the Cassini project can be found on http://www.asp.net/Projects/Cassini/Download
The EULA of Cassini still holds (see the Cassini source files via www.asp.net)

This version of Cassini will allow both local and remote requests.
